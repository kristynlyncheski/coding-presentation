Coding

What is coding?

Coding is the act of writing instructions for computers to make the computer perform a task. It may also be called programming. A finished set of computer instructions is called a program.

Why is coding important?

Every app, game, and software program we use requires a coded program to work. Many of our household appliances and even our cars require a coded program to work.

What is code?

For a program to work, it must be written in computer language. There are different computer languages designed for different types of computer programs. A computer language converts human generated instructions into binary code - strings of ones and zeros that tell a computer what to do.
